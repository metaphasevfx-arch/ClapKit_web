Features

Everything an assistant director needs on set

Project Structure

Project → Day → Scene → Shot → Takes. No chaos and no extra screens.

Action Timer

Precise action duration directly on the workspace screen, linked to each take.

Take States

DAY/NIGHT, INT/EXT, SYNC/MOS, PickUp, plus notes with quick tags.

Camera Tech

Lens, FPS, Shutter, ISO, WB and extended parameters with last-value persistence.

MultiCam

Up to 12 cameras per project, each with its own roll/file and tech settings.

Export and Transfer

Ready-to-share CSV/XLSX reports and project transfer between devices using .clap.

How it works on set

4 steps from first shot to final report

  1. Create a project

    Add production details and default camera settings.

  2. Log takes live

    Timer, states, notes, and technical data are captured in real time.

  3. Save faster

    Take auto-increment, carry-over values, and a clean structured data flow.

  4. Export at wrap

    Generate clean reports and share with the team with no manual rewrite.
